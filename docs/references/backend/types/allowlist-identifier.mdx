# `AllowlistIdentifier`

The `AllowlistIdentifier` class is used to represent an allowlist identifier.

## `AllowlistIdentifier` Properties

| Name | Type | Description |
| --- | --- | --- |
| `id` | `string` | The ID of the allowlist identifier. |
| `identifier` | `string` | The allowlist identifier. |
| `createdAt` | `Date` | The date and time the allowlist identifier was created. |
| `updatedAt` | `Date` | The date and time the allowlist identifier was last updated. |
| `invitationId?` | `string \| undefined` | The ID of the invitation associated with the allowlist identifier. |
