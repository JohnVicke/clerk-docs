# `verifyClient()`

Retrieves a [`Client`](/docs/references/javascript/client) for a given session token, if the session is active.

```tsx
function verifyClient: (token: string) => Promise<ClerkBackendApiResponse<Client>>;
```

## `verifyClient()` Parameters

| Name | Type | Description |
| --- | --- | --- |
| `token` | `string` | The session token to verify. |

## `verifyClient()` Example

```tsx
const token = 'my-session-token';

const { data, errors } = await clerkClient.clients.verifyClient(token);
```
