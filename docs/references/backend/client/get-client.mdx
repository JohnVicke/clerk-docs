# `getClient()`

Retrieves a single [`Client`](/docs/references/javascript/client) by its ID, if the ID is valid.

```tsx
function getClient: (clientId: string) => Promise<ClerkBackendApiResponse<Client>>;
```

## `getClient()` Parameters

| Name | Type | Description |
| --- | --- | --- |
| `clientId` | `string` | The ID of the client to retrieve. |

## `getClient()` Example

```tsx
const clientId = 'my-client-id';

const response = await clerkClient.clients.getClient(clientId);
```
