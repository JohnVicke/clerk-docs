# `getClient()`

Retrieves a single client by its ID, if the ID is valid.

```tsx
function getClient: (clientId: string) => Promise<ClerkBackendApiResponse<Client>>;
```

## `getClient()` Params

| Name | Type | Description |
| --- | --- | --- |
| `clientId` | `string` | The ID of the client to retrieve. |

## `getClient()` Example

```tsx
const clientId = 'my-client-id';

const { data, errors } = await clerkClient.clients.getClient(clientId);
```
