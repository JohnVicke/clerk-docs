# `deleteUser()`

Deletes a user given a valid ID. Throws an error otherwise.

```tsx
function deleteUser: (userId: string) => Promise<ClerkBackendApiResponse<User>>;
```

## `deleteUser()` Parameters

| Name | Type | Description |
| --- | --- | --- |
| `userId` | `string` | The ID of the user to delete. |

## `deleteUser()` Example

```tsx
const userId = 'user_2b8lKxzUgn5M1hsjaTW1mcjc0iB';

const response = await clerkClient.users.deleteUser(userId);

console.log(response);
/* 
{
```
