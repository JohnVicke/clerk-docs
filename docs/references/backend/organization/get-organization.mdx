# `getOrganization()`

Retrieves a single organization by the organization ID, if the ID is valid.

```tsx
function getOrganization: (params: GetOrganizationParams) => Promise<ClerkBackendApiResponse<Organization>>;
```

## `GetOrganizationParams`

| Name | Type | Description |
| --- | --- | --- |
| `organizationId \| slug` | `string` | The ID of the organization to retrieve, or the slug of the organization to retrieve. |

### `getOrganization(slug)`

Retrieve an organization by its slug instead of its ID.

```tsx
const slug = 'my-organization-slug';

const organization = await clerkClient.organizations.getOrganization({ slug });
```
