# `getSession()`

Retrieves a single session by its ID, if the ID is valid. Throws an error otherwise.

```tsx
function getSession: (sessionId: string) => Promise<ClerkBackendApiResponse<Session>>;
```

## `getSession()` Parameters

| Name | Type | Description |
| --- | --- | --- |
| `sessionId` | `string` | The ID of the session to retrieve. |

## `getSession()` Example

```tsx
const sessionId = 'sess_2b6M8I4VZ1wMtgXGQeDe3SJD3cG';

const response = await clerkClient.sessions.getSession(sessionId);
```
